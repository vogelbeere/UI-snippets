Form elements

  • Forms should be accessible.
  • Radio buttons are used for choices where only one item can be selected.
  • Checkboxes are used for choices where more than one item can be selected.
  • Radio buttons and checkboxes must always be associated with a label element. This enables the user to select the choice by clicking on the label as well as the checkbox or radio button.
  • Radio buttons and checkboxes are easier to use than a drop-down.
  • If you have a lot of options to choose from, use a popup lookup, or a scroll panel (<div class="scroll">).
  • Use a checkbox dynamically added by jQuery for security captcha.
  • Provide help text for input fields where appropriate.
