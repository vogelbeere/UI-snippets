Pop-up multiselect tool

See the Pen Popup multiselect by Yvonne Aburrow (@vogelbeere) on CodePen.

 

Usage

If you use the classes and ids provided, the jQuery will iterate through the various form elements and match them with their respective pop-ups.

Use entry-r1, entry-r2 etc as the id for the input fields. Use lookup-r1, lookup-r2 etc as the id for the corresponding popup. Use form-r1, form-r2 etc as the id for the form in the lookup-popup. Use checkboxes-r1, checkboxes-r2 etc as the id for the containing paragraph for the checkboxes in the popup.
